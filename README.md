# erb-types
## Description

This VSCode extension adds support for various flavours of ERB files.

## Currently supported languages:

* JavaScript (ERB)
    - *.js.erb*
* LaTeX (ERB) - *[rails-latex](https://github.com/amagical-net/rails-latex)*
    - *.pdf.erb*
    - **.pf.erbtex*
    - *.tex.erb*
